# Sigma Contábil Digital – Site estático

Site institucional estático de exemplo para um escritório de contabilidade digital, pronto para publicação no **GitHub Pages**.

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Copie todos os arquivos desta pasta para o repositório.
3. Faça commit e push.
4. Em **Settings › Pages**, aponte para a branch `main` e pasta raiz.
5. Atualize `sitemap.xml` e `robots.txt` com a URL final do site.
